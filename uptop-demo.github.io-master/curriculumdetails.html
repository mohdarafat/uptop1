Course Overview

This degree, in collaboration with IIM Rohtak in India, is a 180-hour training course that provides comprehensive coverage of data science with R and Python, along with SQL Programming and data visualization with Tableau. This project-based multi-tool course provides extensive knowledge of the most popular data science techniques used by analytics professional to solve critical business problems.
